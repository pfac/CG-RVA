\abstract{
	Este documento descreve a implementação e mecânica de um jogo usando a biblioteca JSARToolkit para permitir interactividade usando Realidade Aumentada. O JSARToolkit foi usado para construir uma demonstração simples que permite misturar elementos virtuais com imagem em tempo real. O jogo final funciona á custa dos marcadores detectáveis pela biblioteca como local para os monstros aparecerem e interagirem. Foram ainda replicadas algumas das features do jogo original, como os conceitos de captura, evolução, e Pokédex.
}

\additionalabstracttitle{Abstract}
\additionalabstract{
	This document describes the implementation of a game using the JSARToolKit to allow player interactivity using Augmented Reality. Despite the lack of resources, the JSARToolKit was used to build a basic sample which allowed to place virtual elements together with a real-time image. The game was implemented using the markers as place for the monsters to appear allowing interaction by changing the marker visibility. Several features of the original game were replicated, such as the capture system, the evolution and the Pokédex.
}
